# Amazon S3 (`storage.s3`)

**Requires:** `boto3` — `uv pip install "pyconnectors[s3]"`

---

## Configuration

| Key | Description |
|---|---|
| `aws_access_key_id` | AWS access key ID |
| `aws_secret_access_key` | AWS secret access key |
| `region_name` | AWS region (e.g. `us-east-1`) |

---

## Usage

```python
import os
from pyconnectors import ConnectorFactory, ConnectorConfig

config = ConnectorConfig(params={
    "aws_access_key_id":     os.environ["AWS_ACCESS_KEY_ID"],
    "aws_secret_access_key": os.environ["AWS_SECRET_ACCESS_KEY"],
    "region_name": "us-east-1",
})
s3 = ConnectorFactory.create("storage.s3", config=config)

# Upload
result = s3.safe_execute("upload", "my-bucket", "hello.txt", data=b"Hello!")
print("Upload:", result.success)

# Download
result = s3.safe_execute("download", "my-bucket", "hello.txt")
print("Content:", result.data)

# Delete
result = s3.safe_execute("delete", "my-bucket", "hello.txt")
print("Deleted:", result.success)
```
